## CLI Web Scraper

This is a versatile command-line tool written in Go that scrapes news from multiple sources and saves them to filtered text files.

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/zahidayturan/yavuzlar-weekly-tasks.git
cd yavuzlar-weekly-tasks/week-2

```

2. **Install dependencies:**
```bash
go mod tidy

```

### Usage

1. **Run the application:**
```bash
go run main.go

```

- For Help :
```bash
go run main.go -h

```

2. **Follow the menu:** Select a site (1-3) and optionally enter filters like `-date` or `-description` to customize your output file.
